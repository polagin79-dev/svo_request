# svo_request
телеграм бот разрабатывался для приема заявок от бойцов в зоне СВО.

пример заявки

заявка
1) Шурик 12
2) рулетка 11
3) https://ozon.ru/t/if8Gm0S 4

заявки сохраняются в БД
по запросу оператора формировалась таблица(xlsm) с заявками и готовыми сводными таблицами.
Обработанные заявки помечались в БД и повторно в выгрузку не попадали.
Предусмотрено формирование таблицы с заявками за выбранный период, не зависимо были они ранее просмотренны или нет.

команды доступные оператору

/zaiavki

/zaiavki_period

Для вывода данного списка команда необходимо набрать и отправить команду /operator

По назначению бот не использовался.

Тестировалось на Windows10.

Для работы бота необходимо, чтобы рядом с bot.py лежал token.txt, где 
	первой строкой должен быть токен бота (см. token_example.txt).
